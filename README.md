# chess_game
Simple Chess Game using Raylib C++
g++ main.cpp -o main -lraylib -lm -lpthread -ldl -lX11
```
If there are multiple files
```bash
g++ main.cpp Game.cpp Board.cpp Piece.cpp Player.cpp -o chess_game -lraylib -lm -lpthread -ldl -lX11
```
